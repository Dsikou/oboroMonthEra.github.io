# vue 深度作用器

<br/>

- 原因
  - 在做 vue 页面遇到覆盖样式而样式不生效情况经过查找资料得到 深度左右器 深度作用器目的是 选择 vue 原生样式进行修改
  - ">>>"
    - 其作用域是只能选择 class 选择器，无法作用与 id 选择器
    ```
    >>> .el-form-item__label {}
    ```
  - "/deep/"
    ```
    /deep/ .el-form-item__label {}
    ```
  - "::v-deep"
    - 在 vue3.0版本后 替代 "/deep/"
